## Lokaal uitvoeren
Voer `npm start` uit in een terminalvenster. Als de backend ook lokaal gerund zal worden moeten de `environment.ts` url's en de `proxy.conf.json` gecontrolleerd worden op de juiste localhost url's en ports.
